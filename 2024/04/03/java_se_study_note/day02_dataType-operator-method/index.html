JavaSE笔记_Day02：数据类型转换、运算符、方法入门

发表于2024-04-03|更新于2024-04-03

数据类型转换

Java中规定参与运算的值类型一致，如果一致将发生数据类型转换

数据类型大小

由小至大：byte、short、char、int、long、float、double

从左至右：自动类型转换

从右至左：强制类型转换

自动类型转换

将'A'转换成了int类型，值为65

强制类型转换

语法：小数据类型 变量名 = (小数据类型) 值;

补充：计算机中存储数据采用二进制补码

  • 正数

    源码 = 我们直接转换过来的二进制数

    三码合一：源码 = 反码 = 补码

  • 负数

    源码 = 我们直接转换过来的二进制数

    反码 = 源码高位不变，其余取反

    补码 = 反码 + 1

  • int -> byte

ASCII

字符 对应值
0 48
9 57
A 65
Z 90
a 97
z 122

运算符

算术运算符

+ - * / % ++ --

  1. 小数运算：BigDecimal可以解决精度问题

  2. ++ --进行赋值运算时：

    • a = b++; 先赋值，再运算
    • a = ++b; 先运算，再赋值

赋值运算符

= += -= *= /= %=

赋值运算的值要用变量保存

比较运算符

== > < >= <= !=

Java中使用==比较时，要求类型必须一致

基本数据类型用==比对，比对值是否相等

逻辑运算符

&& || !

位运算符

&：有0则0

|：有1则1

^（异或）：相同则0，不同则1

  • a^b^b值不变

~：按位取反

<<：左移。空位补0，被移除的高位丢弃，当运算数类型是byte、short时，将自动扩大为int

>>：右移。被移最高位是0，空位补0，最高位是1，空位补1

>>>：无符号右移。右移后空位补0

三元(目)运算符

语法：变量 = 表达式 ? 值1 : 值2;

方法

定义：对一段重复使用的逻辑进行提取，并取一个名字

注意：方法定义在类中，方法不能嵌套

面试题

&& 和 &，|| 和 | 的区别？

& 和 | 既可以作为逻辑运算符，也可以作为位运算符

&& 和 || 只能作为逻辑运算符，拥有短路操作
