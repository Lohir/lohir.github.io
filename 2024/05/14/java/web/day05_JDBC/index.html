JavaWeb_day05：JDBC

发表于2024-05-14|更新于2024-05-14
|字数总计:1.3k|阅读时长:6分钟|阅读量:

JDBC

概念：使用Java语言操作关系型数据库的一套API

Java操作数据库的流程

  1. 编写Java代码
  2. Java代码将SQL发送到MySQL服务端
  3. MySQL服务端接收SQL语句并执行
  4. 将执行结果返回给Java代码

常用接口

Connection

数据库连接

  • Statement
  • createStatement()

Statement

执行SQL语句的对象

  • int executeUpdate(String sql)：用于执行DDL、DML语句，返回受影响的行数
  • ResultSet executeQuery(String sql)：用于执行DQL语句，返回查询结果的ResultSet

ResultSet

表示查询结果的数据表

  • boolean next()：移动ResultSet的指针到下一行
  • Object getObject(列号或列名)：获取指定列的数据

操作数据库

  1. 获取数据库连接对象

    • 添加MySQL依赖

    • 编写代码

  2. 获取操作数据库语句的对象Statement

  3. 执行SQL语句，返回结果（ResultSet）

  4. 关闭数据库资源

  • 执行DDL或者DML操作

  • 执行DQL操作

注意

不同数据库，驱动不一样

简单提取工具类及其工具类的使用

  1. 在resources目录中新建db.properties文件

  2. 创建一个工具类util

SQL注入问题

开发时，用户输入的内容直接拼接到SQL语句上，改变了SQL语句结构得到非法数据【SQL注入问题，不安全】

解决方式：将Statement使用修改为 PreparedStatement对象即可.

批量操作【了解】

批量插入数据

Statement

  • addBath(String sql)：将SQL添加到批处理的列表中
  • int[] executeBatch()：执行批处理列表中的SQL
  • clearBatch()：清空批处理列表的SQL

连接池

  1. 添加德鲁伊依赖

  2. 在resources目录中新建db.properties文件

  3. 创建一个工具类util
