JavaWeb_day09_day10：HTML + CSS

发表于2024-05-14|更新于2024-05-14
|字数总计:1.8k|阅读时长:8分钟|阅读量:

HTML

介绍

HTML(HyperText Markup Language)：超文本标记语言

W3C标准：规定网页由3部分组成

  • 结构：对应的是 HTML 语言
  • 表现：对应的是 CSS 语言
  • 行为：对应的是 JavaScript 语言

基础标签

  1. HTML结构标签

  2. 标题标签

    h1~h6

  3. <hr/>：横线

  4. <font>：【不建议使用，如有需求，可使用CSS设置】

    • face属性：设置字体
    • size属性：设置文字大小
    • color属性：设置文字颜色

  5. <br/>：换行

  6. <p>：段落标签

  7. `：加粗

  8. `：斜体

  9. <u>：下划线

  10. <center>：居中

图片、音频、视频

  1. <img/>：定义图片
    • src：图像的URL
    • height：图像的高度【?px，?%】
    • width：图像的宽度
  2. <audio/>：定义音频
    • src：音频的URL
    • controls：显示播放的控件
  3. <video/>：定义视频
    • src：视频的URL
    • controls：显示播放的控件

超链接

<a>

  • href：指定资源URL
  • target：指定打开资源方式
    • _self：默认值，当前页面打开
    • _blank：空白页面打开

列表标签

  1. 有序列表

  2. 无序列表

表格

`

：定义表格

  • border：表格边框宽度
  • width：表格的宽度
  • cellspacing：单元格之间的空白

`

：定义行

  • align：定义表格行的内容对齐方式

`

：定义单元格

  • rowspan：规定单元格，可横跨行数
  • colspan：规定单元格，可横跨列数

<th>：定义表头单元格

布局标签

  1. <div>：定义HTML中的一个区域，用来布局网页
  2. <span>：用于组合行内元素

表单标签【重要】

分类

  • <form>：定义表单
  • <input>：定义表单项，通过type属性控制输入形式
  • <lable>：为表单项定义标注
  • <select>：定义下拉列表
  • <option>：定义下拉列表的列表项
  • <textarea>：定义文本域

标签属性

  1. <form>
    • action：表单提交时，表单数据的提交地址，提交地址的URL
    • method：提交表单数据的方式
      • get
        • 默认值
        • 请求参数拼接在URL后面
        • URL长度限制4KB
      • post
        • 浏览器将数据放到http请求消息体中
        • 请求参数无限制
  2. <input>
    • type
      • text：默认值，文本框
      • password：密码框
      • radio：单选按钮
      • checkbox：复选框
      • file：文件上传按钮
      • hidden：隐藏的输入字段
      • submit：提交按钮，会将表单数据发送到服务器
      • reset：重置按钮，清除表单中所有数据
      • button：可点击按钮
    • name：属性名
  3. <select>：下拉列表
  4. <option>：下拉列表项
  5. <textarea>：文本域
  • 以上标签项的内容要想提交，必须得定义 name 属性。
  • 每一个标签都有id属性，id属性值是唯一的标识。
  • 单选框、复选框、下拉列表需要使用 value 属性指定提交的值。

概述

  • CSS：Cascading Style Sheet（层叠样式表）

  • CSS 是一门语言，用于控制网页表现

  • CSS代码定义在<style><style/>标签中

CSS导入方式

    只能作用在一个标签上，复用性差

  • 内部样式：定义<style><style/>标签，在标签内部定义css样式

    可以做到在当前页面复用

  • 外部样式：定义<link>标签，引入外部css文件
