数据类型

分为三类：数值、日期、字符串

  • 数值

    • tinyint：小整数型，一个字节
    • int：大整数型，四个字节
    • double：浮点型，字段名 double(总长度,小数点保留位数)

  • 日期

    • date：年月日
    • datetime：年月日时分秒

  • 字符串

    • char

      定长字符串，声明多少空间就会占用多少空间

      优点：存储性能高

      缺点：浪费空间

    • varchar

      变长字符串，需要多少空间就使用多少空间，但不能超过最大空间

      优点：节约空间

      缺点：存储性能低
